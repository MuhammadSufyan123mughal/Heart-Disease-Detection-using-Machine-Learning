# ❤️ CardioGuard – Heart Disease Prediction System

CardioGuard is a Machine Learning–powered full-stack web application designed to predict the risk of heart disease based on patient health parameters. It combines AI/ML with web development to support early detection and improve healthcare decision-making.

---

## 🚀 Features

- 🧠 Machine Learning-based heart disease prediction
- 📊 Real-time health risk analysis
- 👤 User authentication system
- 🧾 Patient data management
- 📈 Admin dashboard with analytics
- 💬 Interactive chatbot (if included)
- 📱 Fully responsive UI (mobile + desktop)
